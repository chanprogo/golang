
go get 命令可以一键获取代码、编译并安装。

| 参数 | 介绍      |
|:-----| :---------|
| -v  | 显示操作流程的日志及信息，方便检查错误  |
| -d  | 只下载不安装                        |
| -t  | 同时也下载需要为运行测试所需要的包|
| -fix      | 在获取源码之后先运行 fix，然后再去做其他的事情|
| -insecure	| 允许使用不安全的 HTTP 方式进行下载操作|
